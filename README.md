# PanelBuddy

PanelBuddy is a highly customizable tool for displaying desktop pets on your taskbar/panel.

> ⚠️ This project was made for educational purpoes and has a lot of glitches/bugs. The project is still a WIP so don't expect everything working perfectly.
> I highly encourage you to report any issues in the issues tab and i willl try to fix them as soon as i can.

PanelBuddy works using qt6 and is made using pyside6 (qt for python).

## Platforms
The project is meant to work both on windows and linux.

- 🪟 Windows: Works and has been tested
- 🐧 Linux: Due to limitations wayland only works best-effort while X11 works just fine.
